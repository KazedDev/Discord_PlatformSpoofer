## ✦ What is it ?

**PlatformSpoofer** is a [BetterDiscord](https://betterdiscord.app/) plugin that lets you **fake the platform or device** you're using on Discord.

Want to appear as on iOS to get the mobile green dot ? Or show up as on Xbox just for fun ? One click. Done.

---

## ⚡ Features

- 🖥️ **Multi-platform support** — Desktop, Web, Android, iOS, Xbox, PlayStation, VR
- ⚡ **Instant apply** — changes take effect immediately via gateway re-identify
- 🟢 **Floating GUI button** — draggable button always accessible on screen
- 🎨 **Smooth animations** — panel pop, drag trail, drop ripple effects
- 🔁 **Persistent settings** — your chosen platform is saved across restarts
- ⚠️ **Safety warning** — built-in disclaimer so you know the risks

---

## 📦 Installation

1. Make sure [BetterDiscord](https://betterdiscord.app/) is installed
2. Download [`PlatformSpoofer.plugin.js`](https://raw.githubusercontent.com/KazedDev/PlatformSpoofer/main/PlatformSpoofer.plugin.js)
3. Move it to your BetterDiscord plugins folder :

```
%appdata%\BetterDiscord\plugins\
```

4. Open Discord → Settings → Plugins → Enable **PlatformSpoofer** ✅

---

## 🚀 Usage

1. **Look for the green floating button** in the top-right corner of Discord
2. **Click it** to open the platform selection panel
3. **Select your desired platform** from the grid
4. Click **`⚡ Appliquer`** to apply instantly
5. Your Discord status will now show the spoofed platform to others
